This is your electronic notebook. Use it to keep track of:
- What you are doing in the lab.
- What analysis you are running with their results.
- What is said at meetings.
- What theory and models you are learning.
- Etc.

This notebook is good for a single project. Obsidian can be used for multiple projects, your course work, your hobbies, etc. There is a ton of info available on the Internet. 

> [!important]
Your notebook is NOT a collection of all the right answers you found out during your project. It is a map of what you thoughts throughout. **Never, ever, erase any of your notes.** If you realize you made a mistake, make a not of it at the bottom of the erroneous note and write a new one.
## Atomic notes
It's helpful to keep notes contained to a single topic. If on a given day, you read a paper, take some data, and do a bit of analysis on that data, you should probably have at least 3 notes for that day. This way, your notes will be easier to organize and search through later on.
## Tags
To organize the information from your daily notes, we use tags. These go in the "tags" field at the top of each notes. You can create new tags if they are useful. To get started, you will use:
- #data : The note contains information about a data run.
- #analysis : The note contains work on analyzing data.
- #theory : The note contains modeling and theoretical information about your work. Things you have found online, in books, in articles, etc.
- #meeting : The note contains information about what was discussed at a meeting.
## Templates
The templates are all in the "Templates" folder. Do not alter the files in this folder.
### Daily
This template creates a new note in the folder matching the day's date. If that folder doesn't exist, it first creates it. 

To create the new note, press "*Alt+n*" and select "Daily". Enter the tags as necessary and a short description of the note.
### Knit data
When taking data about a knitted sample, there is a lot of information to collect. To help keep track, you can insert the "knit data" template into any note where you are keeping track of the data acquisition. To do so, press "*Alt+e*" and select "knit data". 
### Meeting
This template creates a new note in the folder matching the day's date. If that folder doesn't exist, it first creates it. 

To create the new note, press "*Alt+n*" and select "Meeting". The #meeting tag is automatically applied. Enter additional the tags as necessary and a short description of the meeting.

### Project update
When in a meeting note about the status of your project, press "*Alt+e*" and select to include these standard questions. Answering them will help you situate your work for yourself and for the rest of the group. As you recap your work, you can add links to your notes from that week.
